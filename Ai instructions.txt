MAIN PAGE SETTINGS

MainPage.html has a fixed top-left settings gear.
An account-menu-spacer sits before the header so the settings gear does not hide navigation/content.
The settings dialog is vertically scrollable so all options remain reachable on small screens.

Settings modal layout:
- Settings
- User: current username
- Choose what you want to do on this device.
- Change password section:
  - Original password
  - New password
  - Change password
- Account options:
  - Log out
  - Forget this device
  - Delete account
  - Terms of Use
  - Close

Settings behavior:
- Log out confirms and clears only active session.
- Forget this device confirms and clears active session plus remembered login/offline login/local account cache on this device.
- Delete account confirms twice, requires online access, calls /api/users-delete, deletes the MongoDB account document, and clears local login data.
- Change password requires online access, active session, original password, and new password. It calls /api/users-change-password.
- After password change, remembered same-device login and offline login are updated to the new password.
