Android Install

Install GYMRAT on Android

Chrome can install this website as an Android app. It opens from your Home Screen and can keep working offline after it is cached.

  1. Open this website in Chrome on Android.
  2. Tap the browser menu.
  3. Tap Install app or Add to Home screen.

iPhone Install

Install GYMRAT on iPhone

Open this website in Safari or Chrome, tap Share, then tap Add to Home Screen. If you do not see it right away, tap More or View More and choose Add to Home Screen there.

Desktop Install

Attention, you are about to install GYMRAT on this desktop

Chrome or Edge can install this website as an app on your computer. If the browser install popup is not available, you will see the manual steps.

  1. Open this website in Chrome or Edge.
  2. Click the install icon in the address bar or browser menu.
  3. Click Install.
